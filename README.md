Запускаем сервер 
go run cmd/server/main.go

Открыть в браузере
http://localhost:7540/       (порт задается в .env)

Запуск тестов
go test -run ^TestApp$ ./tests          - Тест для проверки сервера
go test -run ^TestDB$ ./tests           - Тесты для работы с базой данных
go test -run ^TestNextDate$ ./tests     - Тесты для правила повторения задач (функции NextDate)
go test -run ^TestAddTask$ ./tests      - Тесты для добавления задач
go test -run ^TestTasks$ ./tests        - Тесты для списка ближайших задач
go test -run ^TestTask$ ./tests         - Тесты для получения текущего значения
go test -run ^TestEditTask$ ./tests     - Тесты для обновления параметров задач
go test -run ^TestDone$ ./tests         - Тесты для выполнения задач
go test -run ^TestDelTask$ ./tests      - Тесты для удаления задач
go test ./tests                         - Все тесты

Чистим кэш тестов
go clean -testcache

Переменные окружения настраиваем в .env (на данный момент стоят, но такие же как по умолчанию)
